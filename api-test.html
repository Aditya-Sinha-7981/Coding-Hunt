Read-only endpoints — No auth required

GET ?action=status Current round state (poll every 5s) ▶

Returns round number, round type, status (open/closed), poll mode, and event name. No parameters needed.

No parameters required
GET ?action=roundinfo&round=N Full data for a specific round ▶

Returns reference URL + prompt for image rounds, or question + options for poll rounds.

GET ?action=pollresults&round=N Live vote counts for A and B ▶

Returns countA, countB, total. Public endpoint — no auth needed. Call every 3s during polls.

GET ?action=leaderboard&password=XXXX Full ranked leaderboard (password protected) ▶

Returns ranked team list. Must provide admin password from Config tab A2.

Admin endpoints — require adminPassword

POST setControl Change round, status, or poll mode ▶

Send only the fields you want to change. All are optional.

POST revealPoll Reveal secret poll result + trigger scoring ▶

Round must be closed first. Triggers poll scoring immediately.

Requeues error-status submissions so the 60s Gemini trigger picks them up again.
